POST Create Large Order (100+ items)

POST /orders

📋 Outlier Pattern in Action! 🆕

This creates an order with 100+ items. The system will automatically:

  • Set isLargeOrder = true
  • Split items into buckets of 50
  • Store buckets in order_item_buckets collection
🎯 Outlier Pattern

Response

GET Get Order Items

GET /orders/{id}/items

📋 Outlier Pattern Retrieval 🆕

This endpoint transparently retrieves items from both normal and large orders:

  • Normal orders: Returns embedded items
  • Large orders: Retrieves and combines all buckets
